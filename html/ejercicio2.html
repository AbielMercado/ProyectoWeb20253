Definiciones Backend

Backend

Parte de una aplicación web que maneja la lógica del servidor y la base de datos.

API

Interfaz que permite la comunicación entre diferentes sistemas.

Base de Datos

Sistema donde se almacenan y gestionan los datos de una aplicación.

Framework Backend

Conjunto de herramientas y librerias para desarrollar servidores y aplicaciones web.

Servidor

Computadoras o software que maneja peticiones y entrega datos a los clientes.
